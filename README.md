# Dictionary Methods in Python: Student Example

## Dictionary Initialization
A dictionary named `student` is created with keys like `student_id`, `name`, `course`, `topis` (should be `topics`), and `city`.

## Methods Demonstrated

### 1. `clear()`
Removes all items from the dictionary, making it empty.

### 2. `copy()`
Creates a shallow copy of the dictionary.

### 3. `get()`
Retrieves the value for a specified key, e.g., `student.get("topis")`.

### 4. `items()`
Returns a view object of the dictionary’s key-value pairs.

### 5. `keys()`
Returns a view object of the dictionary’s keys.

### 6. `pop()`
Removes a specified key and returns its value, e.g., `student.pop("city")`.

### 7. `setdefault()`
Returns the value of a key if it exists; otherwise, inserts the key with a default value.

### 8. `update()`
Updates the dictionary with another dictionary or key-value pairs.

### 9. `values()`
Returns a view object of the dictionary’s values.

### 10. `popitem()`
Removes and returns the last inserted key-value pair.

## Example Outputs
- Prints the dictionary after each operation to show changes.
- It demonstrates how dictionary methods affect the contents and structure of the dictionary.

This notebook provides a practical overview of Python dictionary methods, useful for learning about dictionaries and their operations.
